## Project Description
This Java-based application is designed to transcribe DNA sequences into RNA sequences. The project is structured using a pure Java approach, ensuring a strong foundation in fundamental programming principles without reliance on external frameworks. Utilizing JUnit for test-driven development (TDD), the project maintains a high standard of code quality and test coverage.

## Prerequisites
To run this project, you need to have the following installed:

- Java Development Kit (JDK) 8 or higher
- JUnit 5: Testing library used to write and run the tests.
- Maven (for dependency management and running tests)
